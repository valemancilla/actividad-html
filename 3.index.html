Receta de Pay de Limón

Pasos principales

  1. Preparar la base de galleta
    1. Triturar galletas
    2. Mezclar con mantequilla derretida
    3. Colocar en el molde y presionar
  2. Preparar el relleno
    1. Licuar leche condensada
    2. Añadir crema de leche
    3. Agregar jugo de limón
    4. Ingredientes extra:
      • Ralladura de limón
      • Rodajas de limón
  3. Verter el relleno sobre la base
  4. Refrigerar el pay

Pasos extra

  1. Decorar con ralladura o rodajas de limón
  2. Añadir crema batida como topping

Tiempo de espera (minutos)

  1. 180 minutos
  2. 120 minutos
  3. 60 minutos
  4. 30 minutos
  5. 15 minutos
